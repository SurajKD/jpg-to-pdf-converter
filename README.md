# JPG to PDF Converter

Privacy-first Next.js app scaffold.

Quick start:
1. Copy this folder to your machine.
2. Run `npm install`.
3. Create `.env.local` from `.env.example`.
4. `npm run dev`.

Choose client-only conversion (recommended) or enable server conversion by setting ENABLE_SERVER_CONVERSION=true and deploying serverless endpoints.

Deploy: Vercel recommended (appDir supported).
